# 🎓 Student Course Registration System

A simple web application built using **Node.js and Express.js** that allows students to **register, login, and view available courses**.  
This project was created as part of a **Cloud Computing assignment** and demonstrates deployment using **Render Cloud Platform**.

---

## 🚀 Technologies Used

- Node.js
- Express.js
- HTML
- CSS
- JavaScript
- Bootstrap
- Git & GitHub
- Render (Cloud Deployment)

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/abhishektech21/student-course-registration.git
```

Go to the project directory:

```bash
cd student-course-registration
```

Install dependencies:

```bash
npm install
```

Run the application:

```bash
node server.js
```

Open the browser:

```
http://localhost:3000
```

---

## ☁️ Deployment

This project is deployed using **Render Cloud Platform**.

Steps followed:

1. Upload project to GitHub
2. Connect GitHub repository with Render
3. Create a Web Service
4. Configure Node.js environment
5. Deploy application
